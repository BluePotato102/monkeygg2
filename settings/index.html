Light/Dark Mode Switch

Switch between Light and Dark mode. Default theme is Dark because Light mode burns my eyes, but apparently people like Light mode so idk here u go.

Auto Cloak

Auto Cloak automatically redirects to another website, closes that website, and opens your page in an #about:blank page. It will clear any trace of you visiting this website in your browser history, unless if you have used this website without cloak before.

Cloak URL

When cloaking, the original page opened will redirect to the URL inserted below. The default is https://mail.google.com (gmail).
